To formalize the compilation of finite deterministic dynamical systems onto physical neuromorphic substrates, we must establish a strict algebraic framework that avoids standard floating-point approximations. We model the underlying deterministic dynamical system as a pair $(X, T)$, where $X$ is a finite state space of dimension $n$, and $T: X \to X$ is the deterministic transition map.

Let $\mathcal{F} = \mathbb{R}^X$ denote the space of all real-valued observables on $X$. The dynamics are captured by the linear, globally defined pullback Koopman operator $K: \mathcal{F} \to \mathcal{F}$, where $Kf(x) = f(T(x))$. #!/usr/bin/env python3
import os
import sys
import hashlib
import subprocess
import requests

ZENODO_API_URL = "https://sandbox.zenodo.org/api/deposit/depositions" 
ACCESS_TOKEN = os.environ.get("ZENODO_ACCESS_TOKEN")
CRATE_ZIP_PATH = "release/AQARION_v2.0_FROZEN.zip"
METADATA_TEX_PATH = "papers/aqarion_metadata.tex"
LATEX_MAIN_FILE = "AQARION_Paper_I.tex"
DOCKER_IMAGE = "texlive/texlive:20240301-minimal"
FROZEN_EPOCH = "1704067200" # Cryptographic time-lock (Jan 1, 2024 UTC)

def calculate_sha256(filepath):
    print(f"[*] Calculating SHA-256 hash for {filepath}...")
    sha256_hash = hashlib.sha256()
    try:
        with open(filepath, "rb") as f:
            for byte_block in iter(lambda: f.read(4096), b""):
                sha256_hash.update(byte_block)
        return sha256_hash.hexdigest()
    except FileNotFoundError:
        print(f"    [!] Error: File not found at {filepath}")
        sys.exit(1)

def reserve_zenodo_doi():
    print("[*] Contacting Zenodo API to reserve DOI...")
    if not ACCESS_TOKEN:
        print("    [!] Warning: No token. Using placeholder DOI.")
        return "10.5281/zenodo.PENDING"
    
    headers = {"Content-Type": "application/json"}
    response = requests.post(ZENODO_API_URL, params={'access_token': ACCESS_TOKEN}, json={}, headers=headers)
    
    if response.status_code == 201:
        return response.json().get('metadata', {}).get('prereserve_doi', {}).get('doi', "ERROR")
    sys.exit(1)

def generate_tex_metadata(zip_hash, doi):
    print(f"[*] Generating {METADATA_TEX_PATH}...")
    tex_content = f"""% Auto-generated by AQARION Binding Script. DO NOT EDIT.
\\newcommand{{\\aqarionVersion}}{{v2.0-FROZEN}}
\\newcommand{{\\aqarionHash}}{{{zip_hash}}}
\\newcommand{{\\aqarionDOI}}{{{doi}}}
\\newcommand{{\\aqarionLeanStatus}}{{100\\% Verified (0 sorry)}}
"""
    os.makedirs(os.path.dirname(METADATA_TEX_PATH), exist_ok=True)
    with open(METADATA_TEX_PATH, "w") as f:
        f.write(tex_content)

def compile_latex_docker():
    print(f"[*] Triggering deterministic PDF compilation via {DOCKER_IMAGE}...")
    papers_dir = os.path.abspath("papers")
    
    docker_cmd = [
        "docker", "run", "--rm", "-i",
        "-e", f"SOURCE_DATE_EPOCH={FROZEN_EPOCH}",
        "-e", "FORCE_SOURCE_DATE=1",
        "-v", f"{papers_dir}:/workdir",
        "-w", "/workdir",
        DOCKER_IMAGE,
        "sh", "-c",
        f"latexmk -pdf -interaction=nonstopmode {LATEX_MAIN_FILE}"
    ]
    
    try:
        subprocess.run(docker_cmd, check=True, capture_output=True, text=True)
        print("    [+] PDF compiled successfully and deterministically.")
    except subprocess.CalledProcessError as e:
        print(f"    [!] Error during LaTeX compilation:\n{e.stdout}\n{e.stderr}")
        sys.exit(1)

if __name__ == "__main__":
    print("====================================================================")

    print("====================================================================")

    reserved_doi = reserve_zenodo_doi()

    compile_latex_docker()

Koopman theory promises linear structure in nonlinear dynamics, but numerical Koopman spectra are easy to compute and hard to trust. A finite EDMD matrix always has eigenvalues; the problem is that many of them may have nothing to do with the infinite-dimensional operator. In this paper we make spectral reliability the objective of dictionary learning. We train neural-network dictionaries not merely to predict the next snapshot, but to minimize Residual Dynamic Mode Decomposition residuals: operator-level a posteriori errors that test whether computed eigenvalues and modes are genuine Koopman spectral objects. To keep the learned observables from collapsing into an unstable coordinate system, the loss also penalizes the condition number of the lifted data matrix. Across conservative and dissipative benchmark systems, the method sharply reduces spectral pollution, improves residual pseudospectral inclusion, and lowers forecast error relative to standard fixed dictionaries. On sea-surface temperature data, it gives cleaner Koopman diagnostics and substantially better one-step forecasts from noisy observations with no governing equations. The message is simple: neural Koopman learning should be judged not by prediction alone, but by whether its spectral claims can be certified. Residuals provide the certificate; conditioning makes it computable.

Liam Llamazares-Elias*, Samir Llamazares-Elias, Jonas Latz, Stefan Klus

Abstract
Global information about dynamical systems can be extracted by analysing associated infinite-dimensional transfer operators, such as Perron–Frobenius and Koopman operators as well as their infinitesimal generators. In practice, these operators typically need to be approximated from data. Popular approximation methods are extended dynamic mode decomposition (EDMD) and generator extended mode decomposition (gEDMD). We propose a unified framework that leverages Monte Carlo sampling to approximate the operator of interest on a finite-dimensional space spanned by a set of basis functions. Our framework contains EDMD and gEDMD as special cases, but can also be used to approximate more general operators. Our key contributions are proofs of the convergence of the approximating operator under relaxed conditions. We also prove that in some cases eigenpairs of the approximating operators weakly converge to eigenpairs of the exact operator, in others they do not. Moreover, we derive explicit convergence rates and account for the presence of noise in the observations. Whilst all these results are broadly applicable, they also refine previous analyses of EDMD and gEDMD. We verify the analytical results with the aid of several numerical experiments.

Matthew J. Colbrook, Igor Mezić & Alexei Stepanenko

Abstract
Many systems resist analytical modeling, making data-driven inference of dynamics important. Yet data-driven methods can fail to converge or generalize, leaving open a central question: When can system behavior be learned reliably from data, and when is such learning impossible? We answer this question using adversarial dynamical systems to identify the boundary between accessible and inaccessible regimes. In Koopman operator learning, a leading framework for representing nonlinear dynamics through linear spectral objects, we design optimal data-driven spectral algorithms with convergence and certification guarantees under conditions arising broadly in physical systems. This yields a convergence theory for Koopman-operator approximations and resolves a longstanding open problem in Koopman spectral analysis. Conversely, by constructing adversarial systems, we prove matching impossibility results: without these conditions, no single-sequence limiting procedure can guarantee learning, regardless of data quality. These results sharply characterize when data-driven spectral learning can succeed and when it must fail. We validate the framework on oscillators, chaotic fluid flows and Arctic sea ice concentration forecasting. In the latter, we uncover hidden modes of Arctic sea ice decline, deliver long-range forecasts with geographic error bounds, and outperform state-of-the-art dynamical and deep learning models at substantially lower computational cost, enabling real-time deployment on standard CPUs.

def calculate_sha256(filepath):  
    """Calculates the SHA-256 hash of a file."""  
    print(f"[*] Calculating SHA-256 hash for {filepath}...")  
    sha256_hash = hashlib.sha256()  
    try:  
        with open(filepath, "rb") as f:  
            for byte_block in iter(lambda: f.read(4096), b""):  
                sha256_hash.update(byte_block)  
        file_hash = sha256_hash.hexdigest()  
        print(f"    [+] Hash: {file_hash}")  
        return file_hash  
    except FileNotFoundError:  
        print(f"    [!] Error: File not found at {filepath}")  
        sys.exit(1)  
  
def reserve_zenodo_doi():  
    """Reserves a DOI on Zenodo without publishing the deposit."""  
    print("[*] Contacting Zenodo API to reserve DOI...")  
    if not ACCESS_TOKEN:  
        print("    [!] Error: ZENODO_ACCESS_TOKEN environment variable is not set.")  
        print("        Falling back to placeholder DOI for local compilation testing.")  
        return "10.5281/zenodo.PENDING"  
      
    headers = {"Content-Type": "application/json"}  
    params = {'access_token': ACCESS_TOKEN}  
      
    # We create an empty deposition just to get the reserved DOI  
    response = requests.post(ZENODO_API_URL, params=params, json={}, headers=headers)  
      
    if response.status_code == 201:  
        data = response.json()  
        prereserved_doi = data.get('metadata', {}).get('prereserve_doi', {}).get('doi')  
        if prereserved_doi:  
            print(f"    [+] Successfully reserved DOI: {prereserved_doi}")  
            return prereserved_doi  
        else:  
            print("    [!] Warning: Deposition created, but prereserve_doi not found in response.")  
            return "10.5281/zenodo.ERROR_EXTRACTING_DOI"  
    else:  
        print(f"    [!] Error contacting Zenodo: {response.status_code} - {response.text}")  
        sys.exit(1)

\subsection{Operator Hygiene and the Pullback Architecture}  
To formalize the compilation of finite deterministic dynamical systems onto physical neuromorphic substrates, we must establish a strict algebraic framework that avoids standard floating-point approximations. We model the underlying deterministic dynamical system as a pair $(X, T)$, where $X$ is a finite state space of dimension $n$, and $T: X \to X$ is the deterministic transition map.  
  
Let $\mathcal{F} = \mathbb{R}^X$ denote the space of all real-valued observables on $X$. The dynamics are captured by the linear, globally defined pullback Koopman operator $K: \mathcal{F} \to \mathcal{F}$, where $Kf(x) = f(T(x))$. Simulating $K$ natively on resource-constrained hardware requires mapping $X$ into a coarse-grained coordinate partition $\Pi = \{B_1, B_2, \dots, B_m\}$ where $m \ll n$. This defines a closed subspace $V_\Pi \subset \mathcal{F}$ of block-constant functions, accessed via the idempotent orthogonal projection operator $P_\Pi$.

Because deterministic dynamics and hardware-imposed projections generally do not commute, we quantify topological leakage via the Global Defect Operator.  
  
\textbf{Definition 1.1 (Global Defect Operator):} For a given dynamics $K$ and partition projection $P_\Pi$, the defect operator $D_\Pi: \mathcal{F} \to \mathcal{F}$ is defined as:  
$$D_\Pi = (I - P_\Pi) K P_\Pi$$

def calculate_sha256(filepath):  
    print(f"[*] Calculating SHA-256 hash for {filepath}...")  
    sha256_hash = hashlib.sha256()  
    try:  
        with open(filepath, "rb") as f:  
            for byte_block in iter(lambda: f.read(4096), b""):  
                sha256_hash.update(byte_block)  
        return sha256_hash.hexdigest()  
    except FileNotFoundError:  
        print(f"    [!] Error: File not found at {filepath}")  
        sys.exit(1)
